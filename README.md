# 🧪 OpenCart Testing Project

## 📌 Project Overview
This project contains QA documentation for testing the OpenCart e-commerce application. It includes Test Plan, Test Scenarios, Test Cases, and Bug Reports to ensure software quality and functionality validation.

---

## 📂 Documents Included

1. Test Plan
   - Scope of testing
   - Objectives
   - Testing types
   - Entry & Exit criteria
   - Resources

2. Test Scenarios
   - Home Page
   - User Registration
   - Login & Logout
   - Product Search
   - Product Details
   - Shopping Cart
   - Wishlist
   - Checkout
   - My Account
   - Contact Us & Compatibility

3. Test Cases
   - 100 detailed test cases
   - Covers functional, UI, and compatibility testing
   - Each test case includes:
     - TC ID
     - Preconditions
     - Steps
     - Expected Result
     - Actual Result
     - Status

4. Bug Report
   - Defect tracking format
   - Severity & priority classification
   - Reproducible steps included
